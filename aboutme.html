Recent Projects

Music Player App

Recently I have developed a Music Player App. It is a cross-platform standalone desktop application developed with Web Technologies. I have used React.js framework. The application connects to Soundcloud API and loads user's playlist. The music player supports pause, play, shuffle, fast forward and rewind. Electron was used to build this cross-platform desktop application. Code for this project can be found at GitHub.
